# 🚀 CosmoCrash

> A Java-based 2D arcade space survival game where players dodge cosmic obstacles, avoid crashes, and chase high scores.
- 🌌 survive as long as possible
- ☄️ avoid crashing into space obstacles
- 🎯 score the highest points
- 🚀 beat your previous best score

---
## 🛠️ Tech Stack
- **Java**
- **Java Swing / AWT**
- **OOP Concepts**
- **Event Handling**
- **Game Loop Logic**
- **Collision Detection**
- **Git & GitHub**

---

## 🎯 Controls
- ASWD
- Space bar to shoot bullets
